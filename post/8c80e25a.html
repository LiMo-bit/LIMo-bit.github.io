存在问题

可能会有反复弹窗的问题，需要执行以下命令，更新pm2版本去解决

  • 更新 NodeJS 版本 > 8.8.0
  • 更新 pm2 到最新版本
    npm install -g pm2@latest
  • 更新内存中的 pm2
    pm2 update

pm2开机自启动

windows设置pm2开机服务自启动nodejs项目 - smile008 - 博客园 (cnblogs.com)

  • 全局安装pm2

    npm instasll pm2 -g

  • 安装windows自启动包

    npm install pm2-windows-startup -g

  • 创建开机启动脚本文件

    pm2-startup install

  • 使用pm2启动项目

    pm2 start xxx

  • 保存pm2中的项目

    pm2 save

    保存之后下次启动将会执行保存的项目

  • 卸载服务，执行：

    pm2-service-uninstall
