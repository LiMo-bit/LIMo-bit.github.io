hexo 小tips

1. hexo修改端口

  • 临时修改端口

    在hexo -server --draft后面添加-p port_num

  • 永久修改端口

    修改node_modules\hexo-server\index.js中的默认端口值

2. hexo永久后台运行

本来想自己整一个服务器，

感觉又很亏，自己记录笔记不如本地，

等把主页弄得差不多了再说租服务器搭博客的事情吧

  • 后台运行hexo

    hexo server --draft &

    由于使用的是windows的git bash终端，所以终端一退就没了

  • pm2后台运行

    hexo 长期后台运行 - eryoung2 - 博客园 (cnblogs.com)

    PM2 - Quick Start (keymetrics.io)

    1. 全局安装pm2

      npm install -g pm2

    2. 在博客根目录下创建脚本

      // title:hexo_run.js
//run
const { exec } = require('child_process')
exec('hexo server --draft', { windowsHide: true }, (error, stdout, stderr) => {
    if (error) {
        console.log('exec error: ${error}')
        console.log('stdout: ${stdout}');
        console.log('stderr: ${stderr}');
        return
    }
})

    3. windows命令行下执行

      cd blog_root_path

      pm2 start hexo_run.js

      PM2实用指南、PM2详解、PM2轻松实现脚本后台执行、Node如何在后台运行、如何让npm start 在后台运行_pm2 如何让命令行在后台运行-CSDN博客

    4. 存在问题

      可能会有反复弹窗的问题，需要执行以下命令，更新pm2版本去解决

      • 更新 NodeJS 版本 > 8.8.0
      • 更新 pm2 到最新版本
        npm install -g pm2@latest
      • 更新内存中的 pm2
        pm2 update

    5. pm2开机自启动

      windows设置pm2开机服务自启动nodejs项目 - smile008 - 博客园 (cnblogs.com)

      • 全局安装pm2

        npm instasll pm2 -g

      • 安装windows自启动包

        npm install pm2-windows-startup -g

      • 创建开机启动脚本文件

        pm2-startup install

      • 使用pm2启动项目

        pm2 start xxx

      • 保存pm2中的项目

        pm2 save

        保存之后下次启动将会执行保存的项目

      • 卸载服务，执行：

        pm2-service-uninstall

  • 数学公式

    Hexo博客使用MathJax公式并解决Markdown渲染冲突问题 - 知乎 (zhihu.com)

    换行不行

    解决mathjax公式不换行问题_mathjax 换行-CSDN博客

    $$
\begin{eqnarray}
\nabla\cdot\vec{E} &=& \frac{\rho}{\epsilon_0} \\\\
\nabla\cdot\vec{B} &=& 0 \\\\
\nabla\times\vec{E} &=& -\frac{\partial B}{\partial t} \\\\
\nabla\times\vec{B} &=& \mu_0\left(\vec{J}+\epsilon_0\frac{\partial E}{\partial t} \right)
\end{eqnarray}
$$

    注意这里使用的换行符是\\\\，因为hexo使用的是mathjax3.x 这个版本两个反斜杠不支持换行
