使用

常用hexo命令

# 清除本地文件
hexo clean

# 本地浏览器预览hexo界面
hexo s

# 部署到git的时候添加commit信息
hexo d -g --message "time@[commit info]"

# 创建草稿
hexo new draft xxx

# 创建发布文章
hexo new post --path xxx/xxx/file_name title

# 发布草稿
hexo publish xxx

TIPS

hexo部署端口修改

  • 临时修改端口
    在hexo -server --draft后面添加-p port_num

  • 永久修改端口

    修改node_modules\hexo-server\index.js中的默认端口值
